VIM/VI notes

Dec 4, 2016 | Hits

在工作当中用到的vi命令记录下来，方便自己随时查阅。后续遇到比较有用的命令也随时更新记录到此。

Common commands

总有一些命令使用非常频繁，记录到文章开头处，便于查找，比如在输入模式下，输入单词的一部分，按ctr+p，vim会自动补全剩下的部分：

cmd descreption cmd descreption
0 line home $ line end
gg doc home G doc end
u undo ctr+r redo
:set nu display line numbers 123gg step to 123 lines
. do last cmd N<cmd> do cmd Nth

Block mode

ctr+V进入列模式，选择目标行之后，可以如下操作：

  • 每一行添加相同的内容：A输入字符串 esc.
  • 代码自动缩进：ctr+V 选中目标行 =.
  • 左右缩进： <左缩进、 >右缩进.

Find the key

查找命令，:set hlsearch可以让查找的关键字高亮显示，:set nohlsearch 则关闭.

-/: 正向扫描查找.
-?: 反向扫描查找.
-n: 跳转到下次匹配.
-N: 跳转到上次匹配.

Move the cursor

normal模式下：

  • 0 : 移动到光标的本行开头.
  • $ : 移动到光标的本行的结尾.
  • H : 移动光标到屏幕的首行.
  • M : 移动光标到屏幕的中间一行.
  • L : 移动光标到屏幕的尾行.
  • gg: 移动光标到文档首行.
  • G : 移动光标到文档尾行.
  • w(e):移动光标到下一个单词.

Copy Paste Delete

y、p、d分别表示复制、粘贴、删除，通常可以与移动光标命令组合使用，在normal模式下：

  • yy : 拷贝光标所在的行.
  • p : 粘贴操作.
  • dd : 删除光标所在的行.
  • yw : 拷贝从当前光标到光标所在单词结尾的内容.
  • dw : 删除从当前光标到光标所在单词结尾的内容.
  • y0 : 拷贝从当前光标到光标所在行首的内容.
  • d0 : 删除从当前光标到光标所在行首的内容.
  • v+选中+y+退出v+p ：需要选中拷贝目标内容到目标位置.
  • 未完待续

尾记

在日常工作过程中，不可避免的要处理一些Linux下的文本编辑。为了一时的方便，总是习惯将Linux文件以samba的方式，交给UE/nodepad++等Windows编辑器处理。其主要原因是对vi/vim编辑器命令不太熟悉，本文目的是在工作当中用到的vi命令记录下来，方便自己随时查阅。后续遇到比较有用的命令也随时更新记录到此。

vim/vi notes

2016.12.04 caopinuo  热度 ℃

在日常工作过程中，不可避免的要处理一些Linux下的文本编辑。为了一时的方便，总是习惯将Linux文件以samba的方式，交给UE/nodepad++等Windows编辑器处理。其主要原因是对vi/vim编辑器命令不太熟悉，本文目的是在工作当中用到的vi命令记录下来，方便自己随时查阅。后续遇到比较有用的命令也随时更新记录到此。

日常很常用的命令记录

总有一些命令使用非常频繁，记录到文章开头处，便于查找：

  • 0 : line home.
  • $ : line end.
  • gg: doc home.
  • G : doc end.
  • u: undo
  • ctr + r: redo
  • :set nu: 显示行号
  • 123gg: 跳转到123行

光标的移动

normal模式下：

  • 0 : 移动到光标的本行开头.
  • $ : 移动到光标的本行的结尾.
  • H : 移动光标到屏幕的首行.
  • M : 移动光标到屏幕的中间一行.
  • L : 移动光标到屏幕的尾行.
  • gg: 移动光标到文档首行.
  • G : 移动光标到文档尾行.
  • w(e):移动光标到下一个单词.

复制 粘贴 删除

y、p、d分别表示复制、粘贴、删除，通常可以与移动光标命令组合使用，在normal模式下：

  • yy : 拷贝光标所在的行
  • p : 粘贴操作
  • dd : 删除光标所在的行
  • yw : 拷贝从当前光标到光标所在单词结尾的内容.
  • dw : 删除从当前光标到光标所在单词结尾的内容.
  • y0 : 拷贝从当前光标到光标所在行首的内容.
  • d0 : 删除从当前光标到光标所在行首的内容.
